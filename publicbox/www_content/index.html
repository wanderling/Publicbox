About PublicBox

PublicBox is specifically designed for community events that have little-to-no internet or phone services available. Using the services provided here you can chat with others, send and recieve files, post events (on the forum), or make and recieve phone calls and text messages using an android program and a service called VoIP (Voice over IP). Completely free and with no strings, ads, or catches.

PublicBox is based on PirateBox

Inspired by pirate radio and the free culture movement, PirateBox is a self-contained mobile collaboration and file sharing device. PirateBox utilizes Free, Libre and Open Source software (FLOSS) to create mobile wireless file sharing networks where users can anonymously share images, video, audio, documents, and other digital content/services.

PublicBox is designed to be safe and secure. No logins are required (except for VoIP) and no user data is logged.
